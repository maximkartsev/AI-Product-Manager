# Landing Page

## User goal

Discover the product and choose an effect to try quickly.

## Key UI

- Hero preview (vertical video)
- Popular effects carousel
- Primary CTA: “Make Yours” (gated by auth when uploading)

## Backend needs

- List effects (active, not deleted)
- Public access allowed for browsing effects

## Acceptance

- Page loads with effects list
- Clicking an effect goes to Effect Detail
- Upload/processing requires auth
